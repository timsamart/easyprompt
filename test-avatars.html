How It Works

  • Deterministic: Same content always generates the same avatar
  • Hash-based: Uses a simple hash function on title + content
  • CSS-only: No external dependencies or image files needed
  • Scalable: Works at any size and resolution
  • Consistent: Looks the same across all browsers and devices
  • Performance: Rendered using pure CSS gradients and patterns
